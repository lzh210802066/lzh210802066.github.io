因为平时记录的笔记在不同的平台发布，后面需要保存成 Markdown 格式又找不到本地的文件记录。  
本地存储的好处很多，比如可以离线浏览阅读、对网页进行备份，也能借助其他工具同步数据以满足自己的工作流程。  
以下这些工具可以帮助你简化操作 



## 一、在线工具推荐

### 1. Devtool

**用法一：**  
十分简单，只需要在左边页面地址处填入网址并按 Enter，就可以在右边看见生成的 Markdown 格式内容，复制使用即可。

**用法二：**  
只需要在左边内容处填入 HTML 代码，就可以在右边看见对应的 Markdown 格式内容，复制使用即可。



### 2. Helloworld

**Helloworld 开发者社区**开源的一个轻量级、强大的 HTML 一键转 Markdown 工具。  
支持多平台文章一键转换，并可将结果保存下载到本地。



## 二、Google 插件推荐

### 1. MaoXian Web Clipper

**MaoXian Web Clipper** 是一个免费的浏览器扩展，可以将网页剪藏到本地硬盘。  
当你想要保存网页时，它能让你“按需裁剪”摘录网页上需要的部分，并将网页剪藏永久保存到本地磁盘。  
这样可以避免网站挂掉、帖子被删、或图片失效等问题。

- 支持浏览器：Chrome、Firefox  
- 免费开源，安装后即可使用。



### 2. Copy as Markdown

**Copy as Markdown** 是一个 Chrome 扩展程序，可将以下内容复制为 Markdown 格式到剪贴板：

- 当前选项卡作为链接  
- 页面上的链接（含图像）  
- 所有打开的标签页作为链接列表  

你还可以自定义快捷键，无需鼠标点击即可复制当前页面链接。

**已知问题：**
- 复制图片时无法抓取 `image.alt` 属性；
- 在 Windows 上复制的页面链接不附带标题。



### 3. 网页转 Markdown

这是一款快速提取网页内容并格式化为 Markdown 的工具，适用于笔记或文章整理。

**主要功能：**
- 一键将网页内容转换为 Markdown 格式；
- 对主流博客、文章类网页效果最佳；
- 支持生成 Markdown 后进行二次编辑；
- 可将生成内容转换为图片保存；
- 支持自定义 Markdown CSS。



 **总结：**  
以上这些工具可以帮助你在不同平台快速获取、保存或转换内容到 Markdown 格式，从而方便本地存档、离线阅读以及多端同步。
